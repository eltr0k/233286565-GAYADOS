Introduction to SQL

SQL (Structured Query Language) is a standard language for managing and manipulating databases.

It is used to perform tasks such as querying data, updating records, and managing database structures.

SELECT * FROM table_name;

SQL is widely used in various database systems like MySQL, PostgreSQL, SQL Server, and more.

Data Types

SQL provides different data types for storing various kinds of data in the database.

Some common SQL data types include:

  • INT – Integer values
  • VARCHAR – Variable-length character strings
  • DATE – Date values
  • FLOAT – Floating-point numbers
CREATE TABLE users (id INT, name VARCHAR(50), birthdate DATE);

Choosing the correct data type is crucial for efficient data storage and retrieval.

CRUD Operations

CRUD stands for Create, Read, Update, and Delete, which are the four basic operations for managing data in a database.

  • Create: Insert new data into a table.
  • Read: Query existing data.
  • Update: Modify existing data.
  • Delete: Remove data from a table.
INSERT INTO users (name, birthdate) VALUES ('John Doe', '1990-01-01');

Each of these operations corresponds to an SQL statement like INSERT, SELECT, UPDATE, and DELETE.

Joins

SQL joins are used to combine rows from two or more tables based on a related column between them.

  • INNER JOIN: Returns records that have matching values in both tables.
  • LEFT JOIN: Returns all records from the left table, and the matched records from the right table.
  • RIGHT JOIN: Returns all records from the right table, and the matched records from the left table.
  • FULL JOIN: Returns all records when there is a match in either table.
SELECT users.name, orders.order_id FROM users INNER JOIN orders ON users.id = orders.user_id;

Joins are essential for querying data that spans multiple tables in a relational database.

Indexes

Indexes in SQL are used to speed up the retrieval of data from a database table.

They work by creating an ordered structure on the columns specified, improving search and query performance.

  • CREATE INDEX: Creates an index on a table.
  • UNIQUE INDEX: Ensures that all values in a column are unique.
CREATE INDEX idx_user_name ON users (name);

Indexes can significantly improve the efficiency of query operations, especially in large datasets.
